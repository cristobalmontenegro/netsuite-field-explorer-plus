# Netsuite Field Explorer Plus

## English

### Synopsis

A Chrome plugin to help you easily discover the Netsuite field names behind a record. This is a fork and enhancement of the original "Netsuite Field Explorer" project by Michoel Chaikin.

![Screen Shot 1](resource/screenshot1.png?raw=true "Screenshot")

### What's New in Netsuite Field Explorer Plus

- **Expand/Collapse All:** You can now expand or collapse all fields with a single click. This allows you to work much faster and view the entire data structure at once.
- **Copy Data Quickly:** A new button allows you to instantly copy the extracted JSON data to your clipboard, giving you higher precision and speed when extracting data from NetSuite.
- **Chrome Manifest V3 Support:** Fully updated and compatible.
- **Works on New UI Records.**

### Features

- One click access to see the field names for NetSuite record fields.
- Colourful tree viewer.
- Quick filtering / search.
- Shows fields that don't come up using the "&xml=T" URL trick.

### Installation

You can load this folder as an "unpacked extension" in your Chrome browser by going to [chrome://extensions](chrome://extensions). 
(The original extension can be found in the [Chrome Webstore](https://chrome.google.com/webstore/detail/netsuite-field-explorer/cekalaapeajnlhphgdpmngmollojdfnd))

### Usage

Simply navigate to a NetSuite record, and click on the extension icon in the toolbar to open the extension. You can define a custom keyboard shortcut by going to [chrome://extensions](chrome://extensions) and clicking Keyboard Shortcuts at the bottom of the page. The default shortcut is `Ctrl + Shift + E`.

## Español

### Sinopsis

Un plugin de Chrome para ayudarte a descubrir fácilmente los nombres de los campos de Netsuite detrás de un registro. Este es un fork y mejora del proyecto original "Netsuite Field Explorer" de Michoel Chaikin.

### Novedades en Netsuite Field Explorer Plus

- **Expandir/Contraer Todo:** Ahora puedes expandir o contraer todos los campos con un solo clic. Esto te permite trabajar mucho más rápido y ver la estructura de datos completa a la vez.
- **Copiar Datos Rápidamente:** Un nuevo botón te permite copiar instantáneamente los datos extraídos en formato JSON a tu portapapeles, brindándote mayor precisión y velocidad al extraer datos de NetSuite. Un arreglo urgente para quienes usan esta herramienta a diario.
- **Soporte para Chrome Manifest V3:** Totalmente actualizado y compatible.
- **Funciona en los Nuevos Registros de la Interfaz de Usuario.**

### Características

- Acceso con un clic para ver los nombres de los campos de registros de NetSuite.
- Visor de árbol colorido.
- Filtrado / búsqueda rápida.
- Muestra campos que no aparecen usando el truco de la URL "&xml=T".

### Instalación

Puedes cargar esta carpeta como una "extensión sin empaquetar" en tu navegador Chrome dirigiéndote a [chrome://extensions](chrome://extensions).

### Uso

Simplemente navega a un registro de NetSuite y haz clic en el icono de la extensión en la barra de herramientas para abrirla. Puedes definir un atajo de teclado personalizado yendo a [chrome://extensions](chrome://extensions) y haciendo clic en Atajos de teclado en la parte inferior de la página. El atajo predeterminado es `Ctrl + Shift + E`.
